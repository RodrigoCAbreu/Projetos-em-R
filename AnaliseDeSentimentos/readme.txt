### Análise de Sentimentos

#Fonte dos dados: Twitter

# Problema de Negócio: Análise de Redes Sociais

#Nesse miniprojeto o objetivo é coletar dados do Twitter, a partir do R, usando técnicas de web scraping. Limpando  os  dados,  organizá-los,  gerar  visualizações e  por  fim  realizar  uma  análise  de sentimentos nos dados coletados, através de técnicas de processamento de linguagem natural e também usando um classificador Naive Bayes.
