### Modelo machine learning - Classificação

#Fonte dos dados: https://archive.ics.uci.edu/ml/datasets/Statlog+(German+Credit+Data)

# Problema de Negócio: Criar  um  modelo  preditivo  para classificar o risco de crédito de clientes de uma instituição bancária

#Este experimento visa demonstrar o processo de construção de um modelo de classificação para prever o  risco  de  concessão  de  crédito  a  clientes  de  um banco. O  dataset  contém 1000observações  e 20variáveis,  representando  osdados  de  clientes,  tais  como:  status  da  conta  corrente,  histórico  de  crédito, quantidade de crédito atual, empregabilidade, residência, idade, etc..
